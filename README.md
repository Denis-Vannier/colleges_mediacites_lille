# Profils sociaux des secteurs de collège de la métropole de Lille

Ce répertoire détaille les traitements réalisés en python sur les fichiers détail du recensement de 2019, afin de déterminer le profils social des parents d'élèves résidant dans chaque secteur de collège. Il s'applique ici aux secteurs de collèges tels qu'ils sont définis en 2022 par le Département du Nord, pour les collèges de la métropole de Lille.
